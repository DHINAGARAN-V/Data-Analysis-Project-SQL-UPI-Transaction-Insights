# UPI Transaction Analysis Using SQL

## Project Overview

This project analyzes UPI transaction data using SQL to understand the growth of digital payments in India. SQL queries are used to explore transaction volume, transaction value, and trends across different banks and time periods.

## Objectives

* Analyze the growth of UPI transactions
* Identify transaction trends over time
* Compare transaction values and volumes
* Extract meaningful insights from transaction data using SQL queries

## Analysis Features

* Transaction volume analysis using SQL queries
* Total transaction value calculations
* Data filtering and grouping using SQL
* Trend analysis using aggregated queries

## Tools Used

* SQL
* MySQL
* CSV dataset
* Data analysis techniques

## Key Insights

* Significant growth in UPI transactions over the years
* Increasing adoption of digital payments
* Trends in transaction volume across different periods

## Files in This Repository

* `upi_transaction_queries.sql` → SQL queries used for analysis
* `dataset.csv` → Raw dataset used for analysis
* `results/` → Query results and analysis outputs
